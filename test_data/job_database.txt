資料庫管理工程師 (Database Administrator / Engineer)

【職位描述】
隨著我們平台流量的爆炸性成長，我們需要一位經驗豐富的資料庫工程師來守護我們的資料核心。您將負責設計、維護與優化我們的關聯式與非關聯式資料庫，確保系統穩定與資料安全。

【主要職責】
- 負責公司內部 MySQL/PostgreSQL 資料庫的日常維運、備份還原與監控。
- 進行資料庫效能調校 (Performance Tuning)，分析與優化慢查詢 (Slow Queries)。
- 規劃與建置資料庫高可用性 (High Availability) 與容災 (Disaster Recovery) 架構。
- 協助開發團隊進行資料庫 Schema 設計與 SQL 語法審查。
- 評估並導入合適的雲端資料庫解決方案或 NoSQL 資料庫 (如 MongoDB, Redis)。

【職位要求】
- 至少 3 年以上 DBA 或資料庫工程師經驗。
- 精通 MySQL 或 PostgreSQL 等關聯式資料庫，熟練掌握各種效能調校技巧。
- 具備大型分散式資料庫或高併發系統維護經驗。
- 熟悉 Linux 作業系統，能編寫 Shell Script 進行自動化維運。
- 對於資料安全與備份策略有深入理解。
- 細心負責，具備良好的抗壓性與問題解決能力。
